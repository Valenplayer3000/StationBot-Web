Introduction

Welcome to Introduction. Now you will learn how to setup the StationBot.



To start, you will need:

  • VS Code (Or other editor)
  • Node.js (For npm and its dependencies)
  • git-scm


After installing. You will need to run following commands:

git --version

node -v

npm -v

Cloning the Repository

If you have checked the version what you did before. Now clone the repository

You can select the version you want to clone:

git clone https://github.com/stationaryStation/stationBot //This is old repository.

git clone https://gitlab.com/stationaryStation/StationBot.git //This is new repository.
